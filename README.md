# SugarCpp
SugarCpp is a language which can compile to C++.

## Examples

#### Hello world
```
void main()
    printf("Hello world!") 
``` 

#### Calculate Sum
```
import "iostream"
       "stdio.h"

int main()
    int sum = 0
    for (int i = 1 to n)
        sum = sum + i
    printf("sum = %d\n", sum)
```
